1. Asm + C 구현되게 만들고
2. 만든 코드를 Library
3. 그걸 DLL로 만들기

의미있는 ASm + C 만들어줘 -> 실행파일을 라이브러리를 사용하여 만들어줘 -> 사용중에 할 수 있는 DLL을 만들어줘 (발표 레포트, QNA, 코드, 코드 간략히 설명)
(ASM + C를 결합한 코드 만들어줘) -> (어셈블리가 포함되어있는 라이브러리 만들어줘) -> (DLL만들고 싶어)

ex) 두 입력받을 변수를 입력하는걸 C언어로만들고, 만약 더하기 프로그램이라면 더하기하는 기능을 라이브러리로 만들고, 메모리 문제로 DLL를 사용해서 메모리 문제 줄이기(필요 없는 코드 지우는 방법)
